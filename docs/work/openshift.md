# OpenShift Security Baseline

**Role:** Platform Architect  
**Year:** 2024  
**Impact:** Enabled compliance-driven workload migrations for 8 enterprise customers

![OpenShift baseline diagram](/assets/project1.svg)

## Summary

Implemented a security-first baseline for OpenShift clusters: pod security standards, network segmentation, RBAC models, and supply-chain enforcement.

## Highlights

- Pod Security Standards and Namespace isolation
- RBAC role matrices and least-privilege constructs
- Image signing, SBOM generation, and CI gates
- Audit logging, alerting and compliance dashboards

## Outcome

Reduced risk surface and enabled migration of regulated workloads with automated policy enforcement and continuous compliance checks.
